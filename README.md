# Mapas OSM (Open Street Map)

---
## Descripción
App android que muestra un mapa de OSM según la ubicación del GPS del móvil.

## Lenguaje/framework

* Lenguaje: Delphi 10.2 Tokyo. 
* Framework: Firemonkey.
